IrcTracker  track(IrcClient client);

Create a new channel and user tracking object for the given irc.client.IrcClient. Tracking for the new object is disabled; use IrcTracker.start to commence tracking.

Bug:
The given client must not be a member of any channels when tracking is started. There is an elegant fix for this, which is planned for the future.

struct  IrcChannel;

Represents an IRC channel and its member users for use by IrcTracker.

The list of members includes the user associated with the tracking object. If the IrcTracker used to access an instance of this type is since stopped, the channel presents the list of members as it were at the time of the tracker being stopped.
