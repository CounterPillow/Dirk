class IrcClient;

Represents an IRC client connection.

this();

Create a new unconnected IRC client.

User information should be configured before connecting. Only the nick name can be changed after connecting. Event callbacks can be added before and after connecting.

Invoked when the requested nick name of the user for this client is already in use.
